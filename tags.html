Different tags

"blockquote" - tag defines a section that is quoted from another source. This source write in attribute cite=""http://www."

"q" - tag defines a short quotation. Browsers normally insert quotation marks around the quotation.


"abbr" tag give useful information to browsers, translation systems and search-engines.

Links

  • Use the "a" element to define a link
  • Use the href attribute to define the link address
  • Use the target attribute to define where to open the linked document
  • Use the "img" element (inside "a") to use an image as a link
  • Use the mailto: scheme inside the href attribute to create a link that opens the user's email program

a href="mailto:job13051987@gmail.com" - відкриває почту, та в новому листі вписує job13051987@gmail.com у полі відправити кому
